Differentiation

tag ความเหมือน ความต่าง

hr & br

 ทำการคั่นข้อมูล hr สำหรับสร้างเส้นคั่น
br สำหรับคั่นข้อความให้ขึ้นบรรทัดใหม่

ul & ol

 เรียงรายการ ul เรียงลำดับสัญลักษณ์
ol เรียงลำดับแบบตัวเลข

select & checkbox & radio button

 ให้ทำการเลือกตัวเลือก select ให้ทำการเลือกตัวเลือกจากรายการ
checkbox เลือกกล่องได้กล่องเดียว
radio button เลือกกล่องได้หลายกล่อง

div & span

 กำหนดลักษณะข้อความ div เรียงต่อในแนวตั้ง
span เรียงต่อในแนวนอน

script & style

 สร้างลูกเล่นให้เว็บไซต์ script ใส่คำสั่ง ่ java script
style สำหรับคำสั่งตกแต่ง css
